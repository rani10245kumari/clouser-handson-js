Question 2

        let count = 0;
        (function () {
          if (count === 0) {
            let count = 1;
            console.log(count); // What is logged?
          }
          console.log(count); // What is logged?
        })();

Output:

1

0

In above code count is globally intialize by value 0. Ater entring to the function there is an condtion if the condtion is true the count will intialize to 1 and then it come out the condtion and both the value will be printed 1 and 0. It is an IIFE (Immediately Invoked Function Expression) function.
